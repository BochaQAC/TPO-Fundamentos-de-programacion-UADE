## ABOUT

Resolucion de problemas de la empresa ficticia "UADEtech corp". El alcance funcional de nuestro proyecto será orientado al desarrollo de un sistema de gestión cinematográfica para la empresa ficticia CineVerse Complex.

El objetivo principal es centralizar la administración de películas exhibidas en el complejo, permitiendo registrar, ver informes, modificar y eliminar información relacionada con la cartelera y el rendimiento comercial de cada película.

## ¿Para qué se va a usar?

Nuestro programa permitirá:

    -Registrar nuevas películas.
    -Modificar datos existentes.
    -Eliminar películas que ya no se encuentren en exhibición.
    -Gestionar el estado de exhibición de cada película.
    -Generar informes generales y reportes para facilitar el análisis de asistencia y rentabilidad.

## ¿Cómo se va a usar?

A través de un sistema de menús interactivos, el usuario podrá navegar entre las distintas opciones disponibles:

1. Alta de películas.
2. Modificación de películas.
3. Baja de películas.
4. Consultas e informes.
5. Salir del sistema.

Cada opción contara con un mensaje el cual indicara como operar correctamente.

## Información administrada

Para cada película se registran los siguientes datos:

    -Título.
    -Género principal.
    -Duración (en minutos).
    -Sala asignada.
    -Cantidad de entradas vendidas.
    -Valor promedio de la entrada.
    -Estado de exhibición.
